== USGS Sparql Party: A SPARQL query visualization application

=== Note: This version is no longer being actively developed.  Please go to the latest version at https://github.com/ColinConduff/GeoSparqlViz3

This web application allows users to analyze and visualize geospatial data.  

The application is available at http://usgs-ybother.srv.mst.edu/geosparql

The original application developed by David Mattli can be found at http://usgs-ybother.srv.mst.edu/viz
